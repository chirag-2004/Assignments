## Viva Questions (Answers)

### 1. What is amortized time complexity?
Average time taken per operation over a sequence of operations.

### 2. Why ArrayList resizing is expensive?
Because a new larger array is created and all elements are copied.

### 3. How LinkedList stores elements internally?
As nodes containing data and references to next (and previous) nodes.

### 4. Difference between singly and doubly linked list?
A singly linked list has one pointer (next), while a doubly linked list has two pointers (next and previous).

### 5. What is space complexity of LinkedList?
O(n), as each node requires extra memory for references.
